# 🌤️ Weather Dashboard



A modern Weather Dashboard that displays **real-time weather information** and a **5-day forecast** using the Open-Meteo APIs. 



## 📌 Features



- Search weather by city

- Smart city suggestions

- Real-time weather information

- 5-Day Weather Forecast

- Dynamic weather icons

- Weather-based animated backgrounds

- Responsive Glassmorphism UI

- Displays:

  - Temperature

  - Humidity

  - Wind Speed

  - Pressure

  - Visibility



## 🛠️ Tech Stack



- HTML5

- CSS3

- JavaScript (ES6)

- Open-Meteo Weather API

- Open-Meteo Geocoding API

## 🚀 How to Run



1. Clone the repository



```bash

git clone https://github.com/Virendra826/Weather-Dashboard/tree/main

```



2. Open the project folder.



3. Open `weather-dashboard.html` in your browser.



> **Note:** Internet connection is required since the application fetches live weather data from Open-Meteo APIs.
